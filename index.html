網站內容檢測

  1. 已設定基本的 favicon
  2. 已使用 PageSpeed Insights 檢測是否有重大問題。
  3. 網站聯絡表單寄件正常，並已正確設定 Reply-To。
  4. 移除 js 與 ajax 程式內的 console.log()，避免 IE 舊瀏覽器發生 JS 錯誤。
  5. 已設定Flash替代顯示內容。(網站內有 Flash 時適用此項。)
  6. 已檢查網站內的 QR code 能否掃描。(網站內有放置 Line, 網址, 聯絡人相關 QR code 時適用此項。)
  7. 已檢查網站內的線上聯絡資訊( Wechat, QQ, skype, FB...)連結是否正確。(網站內有放置前述聯絡方式時適用此項。)
  8. 不使用 embed 語法嵌入影音，以免行動裝置無法顯示。(網站內有嵌入 Youtube 與土豆網…等平台時適用此項。)

網站上線後檢測

  1. 調整 web.config 的CustomError設定，避免直接顯示主機錯誤畫面。
  2. 已檢查是否有失效連結(檢測工具) 。
  3. 已移除 index.html 建構中畫面。
  4. 已檢查 robots.txt 設定檔是否設定正確。
  5. 域名加 www 跟不加 www 都可以連線。
  6. 已將網站相關帳號密碼，如操作後台、簡訊王、Google 帳號、Benchmark 電子報、影音平台...各種帳號記錄存查。
  7. 舊網頁已設定轉址，或是提交搜尋引擎移除。(客戶網站改版時適用此項。)
